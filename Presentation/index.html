  • Star Vector is a full-stack web application.
  • It could be thought of as a hybrid between a social media app and a skills app, but is much more.
  • It aims to bring together programmers seamlessly into a single web app.
  • It creates an efficient market where you can show case your projects and allow others to contribute to your projects

Features

Full Stack

  • Star Vector is fully connected end-to-end with a developed frontend, backend, and database

Monorepo

  • All the code for this entire application, including this presentation, is located in one repository.
  • This enables cross-functional programming as well as simpler building, testing, and deploying.

Tech Stack

Frontend Technologies:

  • React

  • CSS

Backend Technologies:

  • Node

  • Express

  • PostgreSQL

Pages

Homepage

  • Displays the skills that people are using on projects
  • Gives you the ability to visit individual pages and like images.

Search

  • User can search for any projects
  • User can select skills that they want other users to have to contribute to their own project

Profile

  • Users can login and sign up
  • Allows users to projects
